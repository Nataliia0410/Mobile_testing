Windows 10- GitBush - живой девайс
1.Указываем путь - $ export PATH=/c/Users/natal/platform-tools:$PATH
2.Прописываем - adb
3. Проверяем подключенные устройства - adb devices
4. Отобразить логи девайса - adb logcat
5. Остановить отображение логов - ctrl+C
6. Очистить список отображаемых логови - clear
7. Очистить логи всего устройста - adb logcat -c
7. Сохранить логи конкретного мобильного приложения на компьютер в какой то конкретный файлик- adb logcat | grep -rnw "com.android.todolist" > /c/Users/natal/androidAppsNata/logs.log   
          7.1 идем в приложение и делаем шаги    
          7.2 идем в терминал и нажм ctrl-c наш вайл сохраняется файл с логами в файл который мы указали в комманде
==============================================================
Запись логов с эмулятора
1. Запустить андроид студио, выбрать эмулятор, собрать наш apk файл, проверить что логи бегут
2. Терминал GitBush 
      1. Проверяем подключённые устройства - adb devices
      2. Чистим логи устройства - adb logcat -c
      3. Сохранить логи конкретного мобильного приложения на компьютер в какой то конкретный файлик- adb logcat | grep -rnw       "com.android.todolist" > /c/Users/natal/androidAppsNata/logs.log     
      4. Идем на эмулятор и делаем шаги
      5.  чуть чуть подождать - нажать ctrl-c - и сохраняется файл с логами в файл который мы указали в комманде
=====================================
***Командная строка***
1. Установить аpk на реальном девайсе: $ adb install /c/Users/natal/app-debug.apk
                                                      путь на компе где находится файл
2. Cделать скрин экрана через адб (команда) -adb shell screencap /sdcard/scr.png
3. Переместить файл с телефона на комп (Команда адб для переноса файла) -  adb pull /sdcard/screen.png C:/Users/natal/
4. Команда для переноса файла из пк на телефон - adb push D:\Scr_38.png /sdcard/
 **
5.Подключится через WiFi 
      1.Подключаем кабель к телефону > проверяем что телефон подключился как Зарядка к телефону > включаем на телефоне Отладку разработчика > ставим ползунок Отладка по USB ключена > посмотреть в Сведения о телефоне-Состояние-IPдрес > командная строка > adb tcpip 5555  > adb connect (ввести ipадрес который смотрела ранее) 
    Проверяем что все подключилось правильно > adb devices > отображается 2девайся > отключаем телефон кабель > adb devices > отображается девай подключен по ipдресу.
